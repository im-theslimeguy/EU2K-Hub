# About

**EU2K Hub** is a modular digital platform built for the **Európa 2000 Gimnázium** community.
It integrates multiple systems designed to improve communication, organization, and student experience.

The platform brings together dashboards, events, notifications, and collaboration tools into one unified environment.

---

# Main Sections

### Home

Displays the latest school news, announcements, and upcoming events.

### QR-Code

Quick access features designed for mobile users and school-wide interactive activities.

### YouHub

The main **student dashboard**, providing personal insights, class information, schedules, and collaboration tools.

---

# QR-Code Features

* Scan QR codes placed throughout the school
* Participate in school events by finding and scanning hidden QR codes
* Earn rewards during interactive activities
* Transparent permission system that clearly indicates if device peripherals are used

---

# YouHub Features

## My Class

### Class's Month

Monthly performance overview for your class.

* View class progress from up to **8 months in the past**
* See predicted performance for the **next 2 months**

### Class Suggestions

Suggestions and ideas visible only to your class.

* View suggestions submitted by classmates
* Engage with class-specific initiatives

### Information

Important information about your class.

* Current class headcount
* Absence information for the day
* Classroom layout
* Optional public class profile with banner, description, and identity

---

## Calendar

### Class Schedule

* Daily class schedule with times and locations
* View schedules from previous months or predicted upcoming schedules
* Detailed information for each lesson
* Customizable calendar widget with stickers and GIFs
* Notifications before upcoming classes
* Instant alerts when schedule changes occur

### Assignments

* View upcoming assignments for up to **2 months**
* Receive reminders before deadlines
* Collaborate with classmates on group assignments

---

## Notifications

### Personalized Alerts

* Important school updates
* Event reminders
* Schedule changes
* Requests to help with school activities

### Message Notifications

* Instant alerts when receiving messages through **Message Hub**

---

## Suggestions

### Student Suggestions

* View the **4 most recent suggestions**
* Browse all submitted ideas
* Like suggestions to boost visibility

### Submit Your Own

Students can share ideas and prompts with others.

You can submit:

* a title
* the prompt itself
* the outcome or example
* optional tags for easier discovery

### Hive Feedback System

The **Hive** is a digital space where students can:

* submit feedback about the school
* report problems
* suggest improvements or new initiatives

All ideas gather in one place to encourage collaboration and innovation.

> **YouHub helps students visualize progress, stay organized, and share ideas more easily than ever.**
